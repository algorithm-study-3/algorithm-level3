## ✨ 문제 풀이

내장 메서드 Math.max, Math.min에 배열을 넣고 answer에 담아주면 끝

## 코드

```javascript
function solution(s) {
  var answer = "";
  const arr = s.split(" ").map(Number);

  answer = Math.min(...arr) + " " + Math.max(...arr);

  return answer;
}
```

## 📚 정리

- 사실 문자열 안에서도 min, max를 찾을 수 있다. (굳이 Number로 바꿔주지 않아도 됨)

```javascript
arr = s.split(" ");
answer = Math.min(...arr) + " " + Math.max(...arr);
```
