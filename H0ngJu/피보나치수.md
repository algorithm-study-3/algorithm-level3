## 🔗 문제 링크

[피보나치수](https://school.programmers.co.kr/learn/courses/30/lessons/12945)

## 난이도

Lv.2

## ✔️ 소요된 시간

3M

## 코드

```javascript
function solution(n) {
  let a = 0;
  let b = 1;

  for (let i = 2; i <= n; i++) {
    const next = (a + b) % 1234567;
    a = b;
    b = next;
  }

  return b;
}
```
