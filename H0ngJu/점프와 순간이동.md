## ✨ 문제 풀이

0 -> N이 아니라, N -> 0으로 생각하는 것과
N이 짝수냐 홀수냐 케이스를 분리해서 생각하는 것이 핵심.
N이 0이 될 때까지 반복해서 최소 점프 횟수를 구하기

## 코드

```javascript
function solution(n) {
  let answer = 0;

  while (n > 0) {
    if (n % 2 === 0) {
      n /= 2;
    } else {
      n -= 1;
      answer += 1;
    }
  }

  return answer;
}
```
