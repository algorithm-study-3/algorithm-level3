## ✨ 문제 풀이

그냥 인덱스를 반환하면 됨.
for문 돌려도 되지만, indexOf로 쉽게 index를 찾을 수 있음

~~파이썬만 보다가 js로 하려니 어색하다.~~

## 코드

```javascript
function solution(seoul) {
  const index = seoul.indexOf("Kim");
  return `김서방은 ${index}에 있다`;
}
```
