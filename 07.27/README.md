# 📝 알고리즘 문제 풀이 - 2025-07-27

## 문제 📖

- **문제 링크:** [백준 - 하얀 칸](https://www.acmicpc.net/problem/1100)

- **문제 유형:** `구현`

- **난이도:** BRONZE 2

- **풀이 시간:** ⏳ 10분 - 실제 걸렸던 시간

## ✍ 접근 방식

1. 현재 놓여진 체스판을 초기화해서 만들어준다.
2. 기존 체스판을 초기화한다.
3. 정답을 초기화한다.
4. 모든 기존 체스판을 순회하면서 하얀 칸에 체스가 놓인 경우에만 정답을 + 1해준다.
5. 정답을 출력한다.
