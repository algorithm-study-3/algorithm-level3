# 📝 알고리즘 문제 풀이 - 2025-08-01

## 문제 📖

- **문제 링크:** [백준 - 로프](https://www.acmicpc.net/problem/2217)

- **문제 유형:** `디그리디`

- **난이도:** SILVER 4

- **풀이 시간:** ⏳ 10분 - 실제 걸렸던 시간

## ✍ 접근 방식

1. 가장 약한 로프부터 순서대로 오름차순으로 정렬하여 초기화한다.
2. 최대 무게를 0으로 초기화 한다.
3. N만큼 반복을 하고 약한 것을 하나씩 빼면서 최대 무게를 측정한다.
4. 이전 최대 무게와 비교해서 무거운 값을 비교하여 교체한다.
5. 정답을 반환한다.
