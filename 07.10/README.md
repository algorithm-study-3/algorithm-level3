# 📝 알고리즘 문제 풀이 - 2025-07-10

## 문제 📖

- **문제 링크:** [백준 - 수열](https://www.acmicpc.net/problem/2559)

- **문제 유형:** `누적 합`,`두 포인터`

- **난이도:** Silver 3

- **풀이 시간:** ⏳ 30분 - 실제 걸렸던 시간

## ✍ 접근 방식

1. 연속적인 숫자들의 배열을 만든다.
2. max의 초기값을 js에서 표현할 수 있는 최솟값으로 설정한다.
3. K만큼 연속된 일 중 최대 값을 찾는다.
4. 정답을 반환한다.
